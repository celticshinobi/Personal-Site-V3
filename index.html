About Me

I'm not a UX/UI Designer; I'm not a Front-End Developer, nor a Back-End Developer. I don't design pretty graphics and layouts, nor do I write a bunch of code - I do both.

I’m a Web Designer. I'm what was once called a Webmaster. Though, that term seems to have lost some of its importance, you could still call me a Webmaster.

I was born... I grew up (sort of) and I then had the opportunity to use a computer, and my life literally changed. All of the interests that I had up to that point were now gone, and replaced with new interests. I wanted to program and create applications and websites and pictures.

To put things in perspective, I started using the World Wide Web when it was created, and almost as soon as I logged onto the Web, I was programming and creating for it. The Web is not the same as it was then - deep down, it's the same as it always has been, but it's much bigger. Technologies have changed, things have advanced. I've been there through it all, and I plan to be here until my death, which will hopefully be a long time from now.

That all being said, it's time for me to throw some of the usual fluffy industry jargon at you.

Web Designer Industry Jargon

I create Websites and Web Apps using HTML and CSS code. I pride myself on my code being semantic, strict, and valid. I'm extremely passionate about the tiny microscopic details, because even if you don't know - I know that they matter. When those tiny details are stressed over, a mediocre project turns into a brilliant creation.

I also dabble with Javascript, PHP, and ASP.net, and I'm currently digging into Windows 8 App and Game development.

Starving Meteor

I also have a very strong belief that everyone should learn to program, even if it's just a basic understanding of the fundamentals. Programming and creating for the Web is an extremely hot topic right now, because the world has changed - the Web is as important to us as gasoline - or even more important - the Web is tied into nearly everything, whether we realize it or not; our phones, our GPS, businesses, banks, you name it - it's used for everything. It's our job to nurture it and strengthen it.

To that end, I have created Starving Meteor - which, at this point in time is in it's infancy as a Web Design and Development blog, although Its end goal is to do just what the World Wide Web has done since its creation - connect people to knowledge all over the world. Of course, those subjects will be more focused on programming and Web creation - check out the blog here.
